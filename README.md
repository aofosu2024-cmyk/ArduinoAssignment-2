# 7-Segment Countdown with Buzzer

## Introduction
This project demonstrates the use of an Arduino Uno to control a 7-segment display and a buzzer. The system counts down from **9 to 0**, displaying each digit while the buzzer sounds briefly. After reaching zero, the countdown starts again.

## Aim
The aim of this project is to learn how to interface a 7-segment display and a buzzer with an Arduino Uno and program them to perform a countdown sequence.

## Components Required
- Arduino Uno
- 5161AS 7-Segment Display
- Buzzer
- Breadboard
- 220 Ω Resistor(s)
- Jumper Wires
- USB Cable

## How It Works
1. The Arduino starts the countdown from 9.
2. Each number is displayed on the 7-segment display.
3. The buzzer beeps as each number is shown.
4. When the countdown reaches 0, the process repeats continuously.
